ADIF MCP Core

adif-mcp provides a shared Model Context Protocol (MCP) toolkit for ADIF 3.1.5. It defines a safe schema, typed tools, and validation helpers that all adapters (LoTW, eQSL, others) can use.

Features

  • Canonical ADIF 3.1.5 schema mapped into MCP tools
  • Validation and normalization of QSO fields
  • Strict typing + contract-driven queries
  • Reusable by agents, loggers, or downstream services

Roadmap

  • M1 — Core schema & manifest
  • M2 — Integration with LoTW & eQSL adapters
  • M3 — Awards + statistics helpers
